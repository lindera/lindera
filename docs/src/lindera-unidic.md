# Lindera UniDic

Lindera UniDic is a Japanese dictionary crate based on [UniDic](https://github.com/lindera-morphology/unidic-mecab), which uses uniform word unit definitions. UniDic provides more detailed morphological information than IPADIC, with 21 fields per entry.
