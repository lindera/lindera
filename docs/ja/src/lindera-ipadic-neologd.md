# Lindera IPADIC NEologd

Lindera IPADIC NEologd は、新語を含む [IPADIC NEologd](https://github.com/lindera-morphology/mecab-ipadic-neologd) に基づく日本語辞書クレートです。標準の IPADIC 辞書を拡張し、最新の用語や固有名詞を追加した語彙を提供します。
