# Lindera UniDic

Lindera UniDic は、統一的な単語単位の定義を使用する [UniDic](https://github.com/lindera-morphology/unidic-mecab) に基づく日本語辞書クレートです。UniDic は IPADIC よりも詳細な形態素情報を提供し、1エントリあたり 21 フィールドを持ちます。
